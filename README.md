**# IP-threat-intel# Threat Intelligence CLI Tool

A Python-based command-line tool that aggregates threat intelligence for IPv4 addresses using multiple security APIs.

## Features

- Integrates:
  - AbuseIPDB
  - VirusTotal
  - AlienVault OTX
- Concurrent API requests for faster results
- Markdown report generation
- CLI-based workflow
- Modular, object-oriented design
